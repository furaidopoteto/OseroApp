# オセロアプリ (作成日: 2022/2/27)

3つのモードで楽しめるオセロアプリです。

# DEMO

![](demo.gif)

# 特徴

下記３種類のモードがあります。
* CPU(ランダム)

  * ランダムで駒を置くCPUと対戦できます。
* CPU(少し強い)
  * minimax法を使用したCPUと対戦できます。
* OFF
  * 両方の駒を操作できるモードです。

# インストール

```git
$ git clone https://github.com/furaidopoteto/OseroApp.git
```
上記コマンドでダウンロードし「Osero.html」ファイルを開くことで実行できます。
